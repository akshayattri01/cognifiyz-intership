# Cognifyz Technologies - JavaScript Internship

5 JavaScript projects built during internship at Cognifyz Technologies.

## Tasks

### Task 2 - Number Pattern Generator (Beginner)
Generates Pyramid, Inverted Pyramid, Diamond, and Pascal's Triangle patterns using loops.
```bash
node task2_patterns.js
```

### Task 3 - CRUD Task Manager (Intermediate)
Console-based task manager with Create, Read, Update, Delete operations.
```bash
node task3_crud.js
```

### Task 4 - Temperature Converter (Intermediate)
Converts temperatures between Celsius, Fahrenheit, and Kelvin.
```bash
node task4_temp.js
```

### Task 5 - CRUD with File Storage (Advanced)
Enhanced task manager with persistent file storage using JSON.
```bash
node task5_crud_files.js
```

### Task 6 - Web Scraper (Advanced)
Scrapes title, headings, links, images from any website using Axios & Cheerio.
```bash
node task6_scraper.js
```

## Tech Stack
Node.js | JavaScript | Axios | Cheerio
